Welcome! This is the user manual for RedstoneBot.

This short guide will show you how to add and setup RedstoneBot for your Discord server.

Looking for how to create a clone instead? Take a look here.


> Step 1: Add Redstone to your Discord server

Click here to add Redstone to your server. This is the standard webpage for adding bots to Discord servers.

It should look like this:

[image here]

Choose the server you want to add Redstone to and complete the Captcha.

Next, you'll need to give Redstone permission to open and close your server.


> Step 2: Give RedstoneBot permission to control your PloudOS server

Login to PloudOS and access your server list here.

[image here]

Next, choose the server you want Redstone to manage (Note: you must be the owner of your server).

Open the server's management page.

[image here]

Click the "Share Server" tab on the bottom left.

[image here]

Share your server with the user RedstoneBot. Give it permission to open and close your server by selecting the checkboxes. Then, click the "Share button".

[image here]

We're almost there!


> Step 3: Run the setup command

Open Discord on your browser or desktop. Then, go the server you just added Redstone to.

Protip: if you haven't already, you might want to create a separate channel or category just for bots and mute it. Bots are annoying, sometimes.ðŸ˜•

Now, you'll need to link your Discord server to the PloudOS server you want Redstone to control.

All you need to do is run the setup command and specify your server's IP address, like so:

!redstone setup myverycoolserver.ploudos.me

[image here]

If all goes well, you'll get a successful setup confirmation message. If you entered an incorrect/invalid IP, just try again.

If you're sure you're entering a valid IP but didn't get the message in the picture above, then please make sure Redstone has permissions to manage your server, as described in step 2.


> Done! Redstone is ready to roll.

That's all there is to it! To make sure Redstone works, use the status command, like so:

!redstone status

[image here]


> More Features

Take a look at the commands page or use the help command to see all of Redstone's features.

!redstone help
